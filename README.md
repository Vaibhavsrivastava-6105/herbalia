# Herbalia Web App

🌿 **Herbalia** is an intelligent, AI-powered botanical assistant and plant doctor. 
It features a universal plant identifier, an automated disease & health scanner, a curated library of medicines and fertilizers, and a conversational AI agent to help you grow healthier plants.

## Live Demo
🚀 **[Play with the Live App on Vercel!](https://herbaliaweb.vercel.app)**

## Features
- **AI Plant Scanner**: Identify any plant species and detect its health status or diseases instantly.
- **Smart Medicine Recommendation**: Get exact treatment and fertilizer recommendations for diagnosed plant diseases.
- **AI Chat Assistant**: Talk to an expert botanical AI with context-awareness of your scanned images and the medicine database.
- **Curated Botanical Articles**: Read the latest curated insights in plant science.

*Note: API keys are required to run this app locally.*
